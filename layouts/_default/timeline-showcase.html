{{ .Content }} {{/* Get all hacker pages for full timeline */}} {{ $allPages := where .Site.RegularPages "Section" "hacker" }} {{/* Calculate year range for navigation */}} {{- $minYear := now.Year -}} {{- $maxYear := 1990 -}} {{- range $allPages -}} {{- $year := .Date.Year -}} {{- if and (gt $year 1990) (lt $year $minYear) }}{{ $minYear = $year }}{{ end -}} {{- if and (gt $year 1990) (gt $year $maxYear) }}{{ $maxYear = $year }}{{ end -}} {{- end -}} {{/* Generate 5-year interval markers (most recent first) */}} {{- $startMarker := mul (div $maxYear 5) 5 -}} {{- if lt $startMarker $maxYear }}{{ $startMarker = add $startMarker 5 }}{{ end -}} {{- $endMarker := mul (div $minYear 5) 5 -}} {{ if gt (len $allPages) 0 }}
Jump to year: {{- $yearMarkers := slice -}} {{- range seq $endMarker $startMarker -}} {{- if eq (mod . 5) 0 -}} {{- $yearMarkers = $yearMarkers | append . -}} {{- end -}} {{- end -}} {{- range ($yearMarkers | collections.Reverse) -}} {{ . }} {{- end -}}
{{ end }} {{/* Get sample pages for demonstration */}} {{ $samplePages := first 15 $allPages }}

Style 1: Visual Timeline (Current Unified Style)

Left border with dots, grouped by year. Used on hacker profile pages.

{{ range $samplePages.GroupByDate "2006" }}
---- {{ .Key }} ----
{{ range .Pages }}
{{ .Date.Format "2006/01/02" }}
{{ .Params.type | humanize | title }} {{ .Params.conference | humanize | title }}: {{ .Title | markdownify }} {{ range .Params.hacker }} - {{ . | humanize | title }} {{ end }}
{{ end }} {{ end }}

Style 2: Simple List

Clean list format without decorative elements. Compact and scannable.

{{ range $samplePages.GroupByDate "2006" }}
{{ .Key }}
    {{ range .Pages }}
  • {{ .Date.Format "Jan 02" }} {{ .Params.type | humanize }} {{ .Title | markdownify }} @ {{ .Params.conference | humanize | title }}
    • {{ end }}
{{ end }}

Style 3: Card Grid

Each entry as a card. More visual but takes more space.

{{ range $samplePages.GroupByDate "2006" }}
{{ .Key }}
{{ range .Pages }}
{{ .Params.type | humanize | title }} {{ .Date.Format "Jan 2006" }}

{{ .Title | markdownify }}

{{ .Params.conference | humanize | title }}
{{ range .Params.hacker }} {{ . | humanize | title }} {{ end }}
{{ end }}
{{ end }}

Style 4: Compact Table

Tabular format. Best for data-heavy views and sorting.

{{ range $samplePages }} {{ end }}
Date Type Title Conference Speaker(s)
{{ .Date.Format "2006-01-02" }} {{ .Params.type | humanize }} {{ .Title | markdownify }} {{ .Params.conference | humanize | title }} {{ range $i, $h := .Params.hacker }}{{ if $i }}, {{ end }}{{ $h | humanize | title }}{{ end }}

Style 5: Horizontal Timeline

Scrollable horizontal layout. Good for showing progression over time.

{{ range first 10 $samplePages }}

{{ .Params.type | humanize }}
{{ .Title | markdownify | truncate 50 }}
{{ end }}

Style 6: Minimal Dashed (Archive/Retro Feel)

Dashed separators, monospace dates. Terminal/archive aesthetic.

{{ range .Pages }}
{{ .Date.Format "01/02" }} --- [{{ .Params.type | upper | truncate 4 "" }}] {{ .Title | markdownify }} // {{ .Params.conference | upper }}
{{ end }} {{ end }}
